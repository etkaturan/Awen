What Awen does

Every feature built for active practice

Most language apps teach you to recognise. Awen forces you to produce — in writing, in speech, under pressure.

🎙
Speaking practice
Hold the mic, speak German, and receive instant AI feedback on fluency, accuracy, and vocabulary — with specific corrections and B2 exam tips.
Voice + Text modes
🎧
Listening exercises
The AI generates a German text, reads it aloud with a neural voice, hides the text, then asks comprehension questions. Revealed after answering.
5 voice speeds
🔍
Word translator
Look up any German word and get its full breakdown — all meanings, synonyms with usage notes, example sentences, grammar tips, and pronunciation.
Add to flashcards
🃏
Vocabulary flashcards
Build your own word bank with status tracking (new → learning → known). Cards flip to reveal translations. All words persist in a local SQLite database.
Persistent storage
📖
Grammar reference
Six B2 grammar topics — Cases, Konjunktiv II, Konnektoren, Passiv, Wortstellung, Genitiv — with reference tables and an AI explanation button.
AI-powered tips
📄
Paragraph practice
Paste or generate a German paragraph. The AI analyzes it, you reproduce it from memory, and the AI compares both versions with detailed feedback.
4-step flow
How it works

Set up in under 2 minutes

Awen runs entirely on your machine. No account, no subscription. You only need a free Groq API key.

1
Get a free Groq key
Go to console.groq.com and create a free account. Generate an API key — takes 30 seconds. Groq's free tier is generous enough for daily practice.
2
Download and launch
Clone the GitHub repo, run the Python backend with one command, and launch the desktop app. A launch script handles everything automatically on Windows.
3
Start practicing
Enter your key in Settings, pick a topic, and hold the mic button. The AI listens, evaluates your German, and responds — like a real tutor session.
